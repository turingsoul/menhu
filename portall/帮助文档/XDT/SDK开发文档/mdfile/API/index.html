API

API

框架API


属性


version | 【 String 】

dashboard版本


util | 【 Object 】

平台提供的工具函数集合。


lib | 【 Object 】

平台提供的库类。


lib.$ | 【 Object 】

jQuery库，高效精简，功能丰富的JavaScript工具库。


lib._ | 【 Object 】

underscore库，是一个JavaScript使用库，提供一整套函数式编程的实用功能。


widgetBaseClass | 【 Object 】

组件基类，所有组件实现需继承此基类。


方法


bootstrap(cb)

SDK引导程序。

  • 参数

    cb | Function 回调函数

  • 返回值

    【Promise】


register(name,options,isCustom)

注册组件。

  • 参数

    name | 【 String 】

    组件类型

    options | 【Object】

    参数对象 {widgetClass,widgetConfig}

    isCustom | 【Boolean】

    是否自定义组件

  • 返回值

/


add(options)

添加组件

  • 参数

  • options | 【Object】

    参数对象

    options.htmlObj | 【DOM Object】

    组件添加的位置

    options.cfg | 【Object 】

    组件的config对象

  • 返回值

    【Object】组件实例对象


remove(id)

移除组件

  • 参数

    id | 【 String 】

    组件ID

  • 返回值

/

组件基类API


属性


allowQueryErrorCount | 【 Number 】

数据源刷新，最大允许错误次数。


cfg | 【 Object 】

组件配置


cfg.bigandsmall | 【 Boolean 】

是否支持放大/缩小


cfg.chartBtns | 【 Boolean 】

（图表属性）是否显示图表的功能键


cfg.chartDefinition | 【 Object 】

关于图表的定义


cfg.chartDefinition.data | 【 Object 】

数据


cfg.chartDefinition.defaultOption | 【 Object 】

(图表属性)默认option


cfg.chartDefinition.hasFreshQuery | 【 Boolean 】

(组件属性)是否刷新数据源


cfg.chartDefinition.freshQuery | 【 Boolean 】

(图表属性)是否刷新数据源


cfg.chartDefinition.id | 【 String 】

组件配置id


cfg.chartDefinition.inject | 【 Object 】

注入代码


cfg.chartDefinition.option | 【 Object 】

配置option


cfg.chartDefinition.query | 【 String 】

数据源


cfg.chartDefinition.queryid | 【 String 】

数据源id


cfg.chartDefinition.queryname | 【 String 】

数据源名称


cfg.executeAtStart | 【 Boolean 】

执行时启动


cfg.id | 【 String 】

组件id


cfg.layout | 【 Object 】

组件的布局


cfg.listeners | 【 Object 】

被监听的参数


cfg.name | 【 String 】

组件名称


cfg.type | 【 String 】

组件类型


echartsDom | 【 Object 】

echart实例


htmlObj | 【 DOM Object 】

容器


id | 【 String 】

组件id


isEditMode | 【 Boolean 】

是否处于编辑模式


isFirstQuery | 【 Number 】

数据源获取次数。


loader | 【 Object 】

加载图，被创建后有hide与show两种状态


minFreshPeriod | 【 Number 】

数据源刷新最小时间，单位：秒


oldTime | 【 Number 】

上一次组件刷新时间


queryErrorCount | 【 Number 】

数据源错误次数。


queryState | 【 String 】

数据源获取状态。


rawhtmlObj | 【 DOM Object 】

容器


resizeState | 【 string 】

组件实例刷新状态


rootDom | 【 DOM Object 】

容器


tickIsStop | 【 Boolean 】

是否停止刷新。
