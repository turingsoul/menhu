# 辅助系统

### 系统内搜索

当开发的项目越累越多之后，拥有的资源也越来越多，如果想要快速浏览一批名称相似的资源对象，可以使用系统内搜索功能。

![JPG](..\..\img\99.jpg)

系统内搜索是通过名称按词搜索的。

![JPG](..\..\img\100.jpg)

### 消息通知模块

当系统中发生较为重要操作时系统会记录消息到通知模块。未读通知会在操作页头看到红色标记。

![JPG](..\..\img\101.jpg)

如果没有未读通知，铃铛位置则不会有标记。

![JPG](..\..\img\102.jpg)

通过点击铃铛即可进入消息页面。

![JPG](..\..\img\103.jpg)

通过点击通知表头的“√”即可批量标记通知已读。

![JPG](..\..\img\104.jpg)
